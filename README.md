## 진행방식

- **매주 수요일** 스크럼 이후 스터디를 진행합니다.
- 스터디 전까지 2챕터씩 읽은 후 책에 나오는 주제 및 내용을 토대로 각자 예시 코드를 작성해봅니다.
- 코드가 작성되면 main 브랜치에 PR을 날립니다. (+ 각자 정리한 내용이 있으면 추가)
- 스터디 당일 각자 작성한 코드에 대해 설명하고 서로 의논해봅니다.
- 의논된 내용은 PR 코멘트에 정리 후 머지합니다.
- 질문의 경우 issue를 활용합니다.
